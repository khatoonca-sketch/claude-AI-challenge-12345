# Day 4 — Discovering the 90-Day Career Roadmap for BCA Students (AI & Data Focus)

> *"Your journey is a marathon, not a sprint. Stay consistent for 90 days and you will see massive growth."*

---

## What Happened Today

Today was a different kind of day. Instead of just writing code or solving DSA problems, I took a step back and looked at the **bigger picture** — *where am I going, and do I have a plan to get there?*

That's when I came across a **90-Day Career Roadmap built specifically for BCA students** with a focus on AI and Data. And honestly? It changed my perspective completely.

---

## The Roadmap Breakdown

The plan is split into three clear phases:

### 📅 Month 1 — Build Strong Foundations

**Skills to Learn:**
- Python (Basics to OOP)
- SQL Basics (SELECT, JOIN, GROUP BY)
- Git & GitHub (Version Control)

**Projects to Build:**
1. Student Management System (Python)
2. SQL Database Project

**Daily Time Commitment:**
- 1 hour — Python
- 30 minutes — SQL
- 15 minutes — GitHub Practice

> **Focus:** Learn the basics and get comfortable with core tools.

---

### 📊 Month 2 — Data Analytics & AI Basics

**Skills to Learn:**
- NumPy & Pandas
- Matplotlib / Seaborn
- Data Cleaning & EDA
- Introduction to Machine Learning (Scikit-learn)

**Projects to Build:**
1. Sales Data Dashboard (Data Analysis + Visualization)
2. Student Performance Analysis (EDA + Insights)

**Portfolio Goal:** Upload all projects to GitHub with clean code, proper README, and screenshots.

> **Focus:** Work with real data and understand patterns & insights.

---

### 🤖 Month 3 — Machine Learning + Portfolio

**Skills to Learn:**
- Supervised Learning (Regression, Classification)
- Model Evaluation & Metrics
- Feature Engineering
- Basics of Deployment (Flask)

**Projects to Build:**
1. House Price Prediction (Regression)
2. Student Performance Predictor (Classification)
3. AI-powered Web App with Flask (e.g., Resume Screener / Chatbot)

**Professional Branding:**
- Optimize LinkedIn Profile
- Post one learning update weekly
- Connect with students, recruiters, and professionals

> **Focus:** Build ML models, deploy them, and strengthen your online presence.

---

## What I Realized Today

Before finding this roadmap, I was learning **randomly** — a YouTube tutorial here, a random course there. No structure, no end goal.

This roadmap taught me something important:

> **It's not just a learning plan. It's a career plan.**

The three things that hit me hardest:

1. **Projects > Tutorials** — You don't get hired for watching videos. You get hired for building things.
2. **LinkedIn branding is part of the plan** — Not an afterthought. Month 3 literally includes professional presence as a milestone.
3. **Consistency beats intensity** — Even 1 hour of Python + 30 min SQL daily compounds into something massive over 90 days.

---

## Target Outcome After 90 Days

By the end of this roadmap, the goal is to have:

- ✅ 5–6 Strong GitHub Projects
- ✅ Strong Python + SQL + Data Skills
- ✅ Basic Machine Learning Knowledge
- ✅ Professional LinkedIn Presence
- ✅ Internship-Ready Portfolio
